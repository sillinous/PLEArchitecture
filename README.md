# Post-Labor Economics Architecture

> **L/0 — Liberating Humanity from Obligatory Labor**

This repository contains the Community Enterprise Architecture for the Post-Labor Economics movement, founded by David Shapiro.

## 🎯 What is Post-Labor Economics?

Everyone is exhausted. Tang ping, karoshi, quiet quitting, deaths of despair—these are symptoms of a global labor force that has reached the end of its tolerance.

**Post-Labor Economics (L/0)** is a movement and intellectual framework for navigating the transition from labor-based economics to a post-labor society where AI, automation, and robotics perform most work, enabling every human to reach their maximum potential.

### Core Thesis

- **Labor is not sacred** — The belief that work is morally necessary is a historical artifact (Protestant work ethic), not a natural law
- **Automation is inevitable** — Market forces guarantee AI and robotics will displace most human labor
- **Ownership replaces wages** — When labor income disappears, capital ownership must become the primary distribution mechanism
- **Power requires new foundations** — As labor power sunsets, algorithmic power must replace it

## 📐 Key Frameworks

### Pyramid of Prosperity
Five layers for distributing income to households after labor:

| Layer | Name | Mechanisms |
|-------|------|------------|
| 1 | **Universals** | UBI, UHC, Universal Basic Services |
| 2 | **Collectively Owned Public Assets** | Sovereign wealth funds, resource trusts |
| 3 | **Collectively Owned Private Assets** | Cooperatives, ESOPs, DAOs |
| 4 | **Individual Private Assets** | Personal investment portfolios |
| 5 | **Residual Wages** | Protected jobs in meaning economy |

### Pyramid of Power
Five layers for maintaining democratic leverage after labor:

| Layer | Name | Mechanisms |
|-------|------|------------|
| 1 | **Immutable Civic Bedrock** | Blockchain identity, records, property |
| 2 | **Open Payment Rails** | UPI, Pix, CBDC |
| 3 | **Radical Transparency** | Open procurement, transparent budgets |
| 4 | **Direct Democracy** | Participatory budgeting, liquid democracy |
| 5 | **Metagovernance** | Rules for changing the rules |

## 🚀 Quick Start

### Prerequisites
- Ruby 3.0+
- Bundler

### Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# Visit http://localhost:4000
```

### Deploy to GitHub Pages

1. Fork this repository
2. Enable GitHub Pages in repository Settings
3. Select "GitHub Actions" as the source
4. Push changes to trigger deployment

## 📚 Key Concepts

| Term | Definition |
|------|------------|
| **L/0 (Labor Zero)** | Elimination of obligatory human labor; work becomes voluntary |
| **Laborism** | Belief that human labor must be preserved as necessity/virtue—spans left and right |
| **Economic Agency Paradox** | Technology lowers costs but destroys jobs, so no dollars to buy cheaper goods |
| **Algorithmic Power** | Blockchain-based civic infrastructure replacing labor as foundation of leverage |
| **Post-Labor Enterprise** | Corporations operating with minimal/zero human employees |

## 📖 About This Architecture

This site is built on the **Community Enterprise Architecture System (CEAS)**, an open framework for organizations to transparently publish their strategic architecture. It extends traditional Enterprise Architecture (EA) to include:

- Motivation Architecture (Business Motivation Model)
- Stakeholder Ecosystem Mapping
- Capability Maturity Assessment
- Impact Measurement Framework
- Theory of Change
