Quick Reference

Key Data Files

File Purpose
_data/community.yml Core identity: name, mission, vision, values, principles
_data/motivation/ends.yml Goals and objectives (what you want to achieve)
_data/motivation/means.yml Strategies and tactics (how you'll achieve it)
_data/motivation/influencers.yml Internal/external factors, SWOT, risks
_data/stakeholders.yml Stakeholder ecosystem definitions
_data/capabilities.yml Capability map with maturity levels
_data/impact/metrics.yml Impact metrics and theory of change
